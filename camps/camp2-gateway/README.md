# Camp 2: Gateway & Network Security

*"Scaling the Gateway Ridge"*

**Duration:** 95 minutes  
**Primary OWASP Risks:** MCP09 (Shadow MCP Servers), MCP02 (Privilege Escalation via Scope Creep)  
**Secondary Risks:** MCP07 (Insufficient Authentication & Authorization)  
**Azure Services:** API Management (MCP Gateway), Private Endpoints, VNet, API Center  
**Guide Reference:** [microsoft.github.io/mcp-azure-security-guide](https://microsoft.github.io/mcp-azure-security-guide)

## Learning Objectives

- Deploy Azure API Management as a centralized API/MCP Gateway
- Implement JWT validation with audience checking at the gateway
- Configure Private Endpoints for backend network isolation
- Register MCP servers in API Center for governance
- Understand gateway patterns for securing MCP deployments
