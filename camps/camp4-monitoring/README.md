# Camp 4: Monitoring & Telemetry

*"Observation Peak - Seeing Everything"*

**Duration:** 90 minutes  
**Primary OWASP Risks:** MCP08 (Lack of Audit and Telemetry)  
**Secondary Risks:** All MCP01-10 (detection)  
**Azure Services:** Log Analytics, Application Insights, Azure Monitor, Workbooks  
**Guide Reference:** [microsoft.github.io/mcp-azure-security-guide](https://microsoft.github.io/mcp-azure-security-guide)

## Learning Objectives

- Understand why visibility is critical for MCP security
- Configure Log Analytics for centralized MCP telemetry
- Build security dashboards with Azure Workbooks
- Create alerts for suspicious MCP activity
- Detect attack patterns in MCP tool usage
