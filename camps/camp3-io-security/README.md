# Camp 3: Input/Output Security

*"Navigating the Input/Output Security Pass"*

**Duration:** 90 minutes  
**Primary OWASP Risks:** MCP06 (Prompt Injection via Contextual Payloads), MCP05 (Command Injection & Execution)  
**Secondary Risks:** MCP03 (Tool Poisoning)  
**Azure Services:** Azure AI Content Safety, Azure Functions, APIM Policies  
**Guide Reference:** [microsoft.github.io/mcp-azure-security-guide](https://microsoft.github.io/mcp-azure-security-guide)

## Learning Objectives

- Understand prompt injection attacks against MCP tools
- Implement Azure AI Content Safety for content filtering
- Detect and redact PII in tool inputs and outputs
- Validate inputs using JSON Schema
