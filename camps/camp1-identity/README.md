# Camp 1: Identity & Access Management

*"Establishing Your Identity Base Camp"*

**Duration:** 90 minutes  
**Primary OWASP Risks:** MCP07 (Insufficient Authentication & Authorization), MCP01 (Token Mismanagement & Secret Exposure)  
**Secondary Risks:** MCP02 (Privilege Escalation via Scope Creep)  
**Azure Services:** Entra ID, Managed Identity, Key Vault, Container Apps  
**Guide Reference:** [microsoft.github.io/mcp-azure-security-guide](https://microsoft.github.io/mcp-azure-security-guide)

## Learning Objectives

- Understand why identity is the foundation of MCP security
- Implement OAuth 2.1 authentication for MCP clients
- Configure Managed Identity for secure Azure resource access
- Use Azure Key Vault for secrets management
- Apply least-privilege RBAC roles
